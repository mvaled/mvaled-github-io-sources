=========================
 My own password manager
=========================

:date: 2026-03-01
:status: draft
:slug: starch-1password
:category: Programming

I'm an Emacs user and I have stored many of passwords and tokens in ``~/.authinfo.gpg``.  But using
them outside of Emacs is a pain.  Browsers have password managers, there are services like 1Password
and Bitwarden for similar purposes.  But I want just one password manager for all my use cases, and
Emacs is the common denominator.

.. |starch-1password| replace:: ``starch-1password``
.. _starch-1password: https://github.com/mvaled/emacs.lite.d/tree/main/starch-1password

So I joined forces with Opencode and Claude Code to create |starch-1password|_.
